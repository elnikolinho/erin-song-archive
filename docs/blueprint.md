# **App Name**: TuneText

## Core Features:

- Audio Playback: Play .wav audio files in the browser.
- Lyrics Display: Display user-provided song lyrics synchronized with audio playback.
- Audio Upload: Allow users to upload .wav audio files.
- Lyrics Input: Enable users to input and store lyrics for each audio file.
- Lyrics auto-sync: AI tool suggests timestamps for lyrics.

## Style Guidelines:

- Primary color: Vibrant orange (#FF9933) to represent energy and musicality.
- Background color: Soft beige (#F5F5DC) for a calm, music-focused experience.
- Accent color: Muted teal (#3CB371) to complement orange with contrast and tranquility.
- Headline font: 'Belleza', a humanist sans-serif for a stylish design.
- Body font: 'Alegreya', a humanist serif to complement 'Belleza' with good readability.
- Simple, modern icons for play, pause, upload, and settings.
- Smooth transitions for audio controls and lyrics display.
